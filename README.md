# Smart Grievance Box

A complaint management web application developed using Next.js and Django REST Framework.

## Features
- Complaint submission
- Complaint status tracking
- Student/Admin login
- Rating system for resolved complaints
- REST API integration

## Technologies Used
- Next.js
- Django REST Framework
- SQLite
- Axios
- HTML/CSS

## Project Objective
The project helps students submit and track grievances digitally, improving transparency and communication.
